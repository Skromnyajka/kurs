Инструкция по запуску проекта DocFlowSystem

1. Установите Python (если не установлен).
2. Распакуйте архив с проектом.
3. Откройте терминал (командную строку) в папке с проектом.
4. Установите Django командой:
   pip install django
5. Запустите сервер командой:
   python manage.py runserver
6. Откройте браузер и перейдите по адресу: http://127.0.0.1:8000/
